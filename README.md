# FRONTEND FOR SOCIAL APP

## Description
  This is client code for social app using create-react-app.The code for backend api is in separate [repository](hthr)

## Requirements
  The core modules used for this developments are
  ..* create-react-app
  ..* material-ui
  ..* react-router

## Installation

```bash
npm install
```

If you prefer yarn,
```bash
yarn install
```

## Usage
```bash
npm run start
```
